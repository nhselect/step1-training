1. Go to Centre dashboard (in ‘Tracking System’ app) ​

2. Select: Delegates​

3. Search: For the delegate​

4. Select: Edit details ​

5. Update the fields as appropriate ​

6. Select: Save​

> Delegate details are updated and displayed in the registration card​
